# MathSort 📐

> **証明ステップを並び替えて、数学をマスターしよう**

数学の証明を正しい順序に並び替えるパズルゲームです。  
タップ・ドラッグ操作で証明ステップを並べ、定理の構造を楽しく学べます。

🔗 **プレイはこちら**: https://math-sort.vercel.app

---

## 🎮 ゲームの遊び方

1. **カテゴリ / 難易度を選ぶ**  
   「ランダムに挑戦」または「カテゴリから選ぶ」でステージを開始します。

2. **証明ステップを並び替える**  
   シャッフルされた証明ステップを正しい順序に並べましょう。
   - **タップ** → 選択肢エリア↔答えエリアを即座に移動
   - **長押し → ドラッグ** → 細かい順序を調整

3. **自動判定 & スコア**  
   全ブロックが答えエリアに揃うと自動で正解判定。ヒントを使わず解けると **S ランク**！

4. **連続チャレンジで streak を伸ばそう**  
   毎日解いてストリーク（🔥 日連続）を記録しましょう。

## ✨ 主な機能

| 機能 | 説明 |
|------|------|
| 📐 LaTeX 数式レンダリング | KaTeX による美しい数式表示 |
| 📱 タップ移動 | ブロックをタップするだけで上下エリア間を瞬時に移動 |
| 👆 長押しドラッグ | 長押しでドラッグ＆ドロップに切り替え、細かく順序を変更 |
| 📌 ピン留めブロック | 定理の宣言文など固定行は自動配置済みで移動不可 |
| 💡 ヒント機能 | 最大 3 つのヒントで詰まっても安心（ヒント数でスコアに影響） |
| 📖 解説・正解スワイプ | 正解後に「解説」→スワイプ→「正解の証明」の2ページ構成で表示 |
| 🔥 ストリーク | 毎日クリアすることで連続日数を記録（GitHub アカウント連携で同期） |
| 🏆 スコアリング | S / A / B / C ランクで達成度を評価 |
| 📱 スマホ最適化レイアウト | 定理・証明欄を別ページに分け、スワイプで切り替え可能 |

## 🧩 問題の追加方法

1. **問題ファイルを作成**  
   `js/data/<カテゴリ>/<問題ID>.js` を作成し、以下の形式で記述します。

```javascript
(window.PROBLEMS_REGISTRY = window.PROBLEMS_REGISTRY || []).push({
  id: 'stat_001',
  title: '標本分散の不偏性の証明',
  category: 'statistics',
  difficulty: 3,
  language: 'proof',
  description: '【定理】... $E[S^2] = \\sigma^2$ を証明せよ。',
  pinnedCode: ['【証明】'],  // 固定行（定理の宣言文など）
  blocks: [
    { id: 0, code: 'まず $\\sum(X_i - \\bar{X})^2$ を展開する。' },
    { id: 1, code: '$= \\sum X_i^2 - n\\bar{X}^2$' },
    // ...
  ],
  // 部分順序制約（[a, b] = 「ステップa はステップb より前」）
  partialOrder: [
    [0, 1], [1, 2],
  ],
  hints: ['ヒント1', 'ヒント2'],
  explanation: {
    summary: '解説の概要',
    points: ['ポイント1', 'ポイント2'],
    complexity: { time: '計算の性質', space: '必要な知識' },
    tip: '学習アドバイス',
  },
});
```

2. **`index.html` に script タグを追加**（`registry.js` の読み込みより前に追加）

3. **数式は `$...$`（インライン）/ `$$...$$`（ディスプレイ）** で記述すると KaTeX が自動レンダリングします。

---

## 🔧 正解判定について

- **`partialOrder` 方式**（推奨）: `[a, b]` の組みで「ステップ a はステップ b より前」という制約を指定。論理的に等価な複数の証明順序を自動で正解と判定します。例えば $E[X_i^2]$ と $E[\bar{X}^2]$ の計算は互いに独立であり、どちらが先でも正解です。
- **`correctOrders` 方式**: 正解の並び順を配列で直接指定します。
